◆ get_history()

history.get_history ( dict current_user = Depends(get_current_active_user))

Busca todo o histórico de conversas do usuário logado.

Verifica autenticação automaticamente via Depends, busca histórico usando email do usuário como chave, converte datetime para string e retorna histórico formatado.

Parâmetros
current_user(dict) Dados do usuário autenticado
Retorna
(dict) Histórico de conversas com total de entradas e dados do usuário
