◆ create_documents_from_text_manual()

list[dict] utils.create_documents_from_text_manual ( str text,
str filename,
int chunk_size = 1000,
int overlap = 200 )

Implementa um divisor de texto customizado para segmentar documentos longos em chunks menores.

Inclui sobreposição entre chunks para preservar o contexto. Essa abordagem substitui a necessidade de bibliotecas de text splitting mais pesadas como Langchain.

Parâmetros
text(str) Conteúdo textual completo do documento
filename(str) Nome do arquivo original para metadados dos chunks
chunk_size(int) Tamanho máximo desejado para cada chunk
overlap(int) Número de caracteres de sobreposição entre chunks consecutivos
Retorna
(list[dict]) Lista de dicionários com 'content' e 'metadata' de cada chunk

◆ simple_search()

list[dict] utils.simple_search ( str query,
list[dict] documents,
int max_results = 3 )

Sistema de busca por relevância baseado em contagem de palavras-chave.

Implementa scoring simples mas eficaz para recuperação de informações. Funciona como fallback quando o Pinecone não está disponível.

Parâmetros
query(str) Pergunta do usuário para a busca
documents(list[dict]) Lista de documentos (chunks) no formato {'content': '...', 'filename': '...'}
max_results(int) Número máximo de resultados a serem retornados
Retorna
(list[dict]) Lista de documentos relevantes ordenados por score

◆ simple_text_splitter()

list[str] utils.simple_text_splitter ( str text,
int chunk_size = 1000,
int overlap = 200 )

Implementação alternativa de divisor de texto para otimizar processamento de documentos grandes.

Divide texto em chunks com sobreposição para manter contexto entre segmentos. Retorna apenas o conteúdo dos chunks (sem metadados).

Parâmetros
text(str) Texto a ser dividido em chunks
chunk_size(int) Tamanho máximo de cada chunk
overlap(int) Sobreposição entre chunks para preservar contexto
Retorna
(list[str]) Lista de strings contendo os chunks de texto
